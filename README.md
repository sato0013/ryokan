# ryokan-project

## 概要

九州の一人でも行きやすい温泉旅館に特化したサイトになっています。

管理者ユーザと一般ユーザに切り分けてログインできるようにしています。

## 使い方

管理者ユーザは温泉旅館を写真付きで投稿することができます。

一般ユーザは投稿された温泉旅館を九州の県別に閲覧でき、マイページからいいねした旅館をまとめて見ることができます。
